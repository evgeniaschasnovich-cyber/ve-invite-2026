# Свадебный лендинг

Статичный адаптивный лендинг по референсу из изображения.

## Как запустить

Откройте `index.html` в браузере или загрузите папку на любой хостинг.

## Что заменить перед публикацией

1. В `script.js` замените `your-email@example.com` на email для RSVP.
2. В `index.html` замените ссылку кнопки «Нажать для вдохновения» на Pinterest/Telegram/галерею.
3. При необходимости замените ссылку Google Maps у кнопки «Открыть карту».
4. Текущая дата свадьбы для таймера: `2026-08-07T16:00:00+03:00`.
